## 市場關鍵趨勢（2026年）

1. **成交熱區**：AI 客服+LINE Bot $15K~$50K（量最大）、RAG 知識庫 $80K~$300K（利潤最好）、n8n 自動化 $15K~$50K（門檻低）
2. **價格走勢**：整體持平微降 5-10%（工具成熟、供給增），AI Agent 逆勢漲 20-30%（供不應求），純 API 串接明顯下跌
3. **客戶真實預算**：多數中小企業 5~30 萬（遠低於坊間寫的百萬級），「預算詳談」佔約 40%
4. **競爭**：每案平均 30-50 人提案；純串接案競爭最激烈（50+人）；有 domain knowledge 的 AI 案競爭少很多
5. **真正賺錢模式**：不是一次性開發，而是月維護費（開發費 15-25%）+ 模型 API 代管
6. **年維護費**：RAG/客服約開發費 15-25%、fine-tuning 約 25-35%
