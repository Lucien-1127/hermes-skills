# FileOps Skill v2.0

## 用途
檔案組織、重複刪除、備份管理、存儲最佳化。

## 安全等級（遞增的操作權限）

| 等級 | 操作 | 確認方式 |
|------|------|--------|
| L1 | 讀取、掃描、報告 | 無需確認 |
| L2 | 移動、複製、壓縮 | 自動執行 + 記錄 |
| L3 | 刪除、覆寫 | **必須手動確認** |

## 整理任務
1. **重複檔案掃描**
   - MD5 hash 比較，找出完全重複
   - 保留原始，移除副本到 ~/.trash/

2. **舊檔案歸檔**
   - 未訪問 >90 天 → 壓縮到 ~/Archive/
   - 記錄路徑以備恢復

3. **資料夾組織**
   - 按副檔名分類（~/Downloads/ → ~/Documents/, ~/Images/, etc.）
   - 空資料夾刪除

4. **備份驗證**
   - 掃描所有重要檔案（~/Documents/, ~/.ssh/, etc.）
   - 檢查備份存在性與新鮮度

## 執行步驟
1. L1 掃描（無風險）：列出所有操作候選
2. 使用者確認清單
3. L2/L3 操作（建立檔案清單，移動/備份先行）
4. 驗證完整性（如刪除，進行快照驗證）
5. 記錄操作日誌

## 禁止事項
- ❌ 無詢問刪除檔案（即使 L2 也要記錄，未來可審計）
- ❌ 跨越 L3 邊界無明確授權
- ❌ 刪除系統關鍵檔案（~/.ssh/, 設定檔等）
