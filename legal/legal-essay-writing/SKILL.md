# 法律申論題寫作範本

## 核心答題架構

### IRAC（英美法系）／ 三段論法（大陸法系）

```
Issue   （爭點）  → 「本案爭點為…」
Rule    （法規）  → 「按民法第…條規定…」  [大前提]
Application（涵攝）→ 「本件事實…符合前開要件，蓋…」 [小前提]
Conclusion（結論）→ 「綜上，本件應…」 
```

### 台灣國考標準格式層次

```
壹 >> 一 >> (一) >> 1. >> (1) >> ①
```

## 題型分類與範本

### 1. 法條適用題 — 給情境，問適用哪條法

**範例開頭：**
> 「小明賣了依託咪酯給小花，毒品危害防制條例第幾條管得到他？」

**答題結構：**
```
壹、甲之行為可能構成毒品危害防制條例第X條之罪
一、按毒品危害防制條例第X條規定：「…」
    → 大前提
二、本件事實中，甲…（涵攝事實至法條要件）
    → 小前提
三、結論：甲之行為該當本罪
```

### 2. 判例分析題 — 給判決摘要，問見解

**範例開頭：**
> 「最高法院曾有判決認為…你覺得這個見解在今天的案子適用嗎？」

**答題結構：**
```
壹、最高法院OO年度台上字第XX號判決見解分析
一、該判決之案例事實與見解要旨
二、本案與判決事實之異同比較
三、該見解於本案適用之可行性
```

### 3. 爭點歸納題 — 給複雜案情，問爭點

**範例開頭：**
> 「這個案子牽涉到好幾個問題，你認為最關鍵的爭點是哪個？」

**答題結構：**
```
壹、本案爭點分析
一、爭點一：…（層層拆解）
二、爭點二：…
三、爭點三：…
貳、綜合評析
```

### 4. 策略題 — 問訴訟或答辯方向

**範例開頭：**
> 「如果你是小明的律師，你第一步會怎麼打？」

## 批改標準（給分架構）

| 項目 | 配分 | 評分重點 |
|------|:----:|---------|
| 法條正確性 | 40% | 引用條號正確、條文內容正確 |
| 爭點掌握 | 30% | 是否抓到核心爭點 |
| 論理邏輯 | 20% | 推論是否合理、有層次（涵攝） |
| 表達完整性 | 10% | 是否完整回答問題 |

## 科目特色

### 刑法
- 三階層體系：構成要件該當性 → 違法性 → 罪責
- 關鍵：客觀歸責、主觀故意/過失區分
- 競合論（想像競合 vs 數罪併罰）

### 民法
- 請求權基礎思考法：誰 → 向誰 → 依何請求權 → 有無阻卻事由
- 契約、侵權、不當得利、物權四路徑
- 時效抗辯不可漏

### 憲法
- 基本權利：保障範圍 → 限制 → 合憲性審查（比例原則）
- 權力分立：組織、權限、制衡
- 釋字/憲判字號引用

### 行政法
- 行政處分 → 訴願 → 行政訴訟
- 不確定法律概念與裁量
- 判斷餘地與司法審查密度

## 時間分配（以60分鐘為例）

| 階段 | 時間 | 說明 |
|------|:----:|------|
| 審題 | 10 min | 讀題、圈關鍵字、畫爭點 |
| 擬答大綱 | 5 min | 寫大綱、配分 |
| 寫作 | 40 min | 依大綱寫作 |
| 檢查 | 5 min | 錯字、條號、邏輯缺口 |

## 常見扣分陷阱

- ❌ 條號寫錯（如 §4 寫成 §5）
- ❌ 未區分既遂與未遂
- ❌ 忽略證據能力問題
- ❌ 事實涵攝不足（只寫法條沒套到事實）
- ❌ 引用已廢止法條
- ❌ 結論空白或矛盾

## 長期提升策略

1. 每週至少練兩題完整申論題
2. 練習時限時（培養時間壓力下的答題能力）
3. 寫完後自我批改，對照給分標準
4. 累積自己的判決資料庫（重要見解+條號）
5. 大題小作 vs 小題大作：視配分調整篇幅

## 🔴 強制：引用驗證流程（Zero-Hallucination）

所有法條引用與判決引用**必須即時聯網驗證**，不得依賴 LLM 記憶。

法規代碼與 URL 速查表見 `references/citation-verification-urls.md`。

### 法條引用驗證

每條引用必須依序執行以下三步：

**Step 1 — 查全國法規資料庫**
```
https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode={法規代碼}&flno={條號}
```
- 確認條號存在、條文內容正確
- 確認該條未被刪除或修正

**Step 2 — 查法規現狀對照表**
zhiyan-legal 的 `docs/60_概念詞條/法規現狀參考表.md` 記錄已知法規更新狀態：
- 毒品類：每週更新
- 勞動/公司/會計類：每月更新
- 穩定條文：每季更新
- 過期條目必須聯網重新確認

**Step 3 — 產出驗證結果**
```
📋 法條引用檢查
├── ✅ 毒品危害防制條例第4條第1項 — 正確
│    └── 🔗 https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000008&flno=4
├── ⚠️ 引用刑法第271條 — 本案為毒品案，不適用殺人罪
│    └── 扣 2 分
└── ❌ 引用已廢止條文 — 該條已於 YYYY/MM/DD 刪除
     └── 扣 5 分
```

### 判決引用驗證

每筆判決引用必須聯網查司法院判決書查詢系統：

**查詢端點：**
```
https://judgment.judicial.gov.tw/
```
搜尋案號（如 `最高法院112年度台上字第XXX號`），確認：
- ✅ 案號存在，內容與引用一致
- ⚠️ 案號存在但見解與引用有出入
- ❌ 案號不存在或已遭推翻

### 引用錯誤處理

| 情況 | 處理方式 |
|------|---------|
| 條號正確、條文有效 | ✅ 正常引用，附來源連結 |
| 條號正確但條文已修正 | ⚠️ 引用最新條文，標示修正日期 |
| 條號不存在 | ❌ 整條刪除，告知正確條號 |
| 判決案號正確 | ✅ 正常引用，附查詢連結 |
| 判決已遭推翻 | ⚠️ 標示「本判決已遭 OOO 號判決推翻」 |
| 判決不存在（幻覺） | ❌ 整段刪除，扣分 |

### 免責聲明強制標示

所有申論題批改報告結尾**必須**附上：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚠️🚨 重要免責聲明 🚨⚠️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
本申論題測試僅供教育訓練用途，不構成法律意見。
法條引用已盡力驗證，如有疑義請以全國法規資料庫為準。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 驗證優先序

```
LLM 產出引用
    ↓
聯網查全國法規資料庫（law.moj.gov.tw）
    ↓ 條號正確？
    ├── 是 → 聯網查司法院判決書查詢（judgment.judicial.gov.tw）
    │         ↓ 判決存在？
    │         ├── 是 → ✅ 輸出，附來源連結
    │         └── 否 → ❌ 標示幻覺，刪除
    └── 否 → ❌ 標示錯誤，以現行法為準
```

**原則：**
- 聯網驗證是 mandatory，不是 optional
- 無法驗證的引用 → 寧可不寫，不可編造
- 每條引用附來源連結，供使用者自行查核
