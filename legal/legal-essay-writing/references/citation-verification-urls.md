# 法條引用驗證 URL 速查

## 全國法規資料庫（法條驗證）

通用格式：
```
https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode={法規代碼}&flno={條號}
```

常用法規代碼：
| 法規 | 代碼 |
|------|------|
| 民法 | B0000001 |
| 刑法 | C0000001 |
| 刑事訴訟法 | C0010001 |
| 民事訴訟法 | B0010001 |
| 憲法 | A0000001 |
| 毒品危害防制條例 | C0000008 |
| 藥事法 | L0030001 |
| 著作權法 | J0070017 |
| 勞動基準法 | N0030001 |
| 公司法 | J0100001 |

## 司法院判決書查詢（判決驗證）

端點：
```
https://judgment.judicial.gov.tw/
```

搜尋欄位：案號（例如：`最高法院112年度台上字第XXX號`）

## 引用驗證流程

```
Step 1: 查全國法規資料庫確認條號有效
Step 2: 查司法院判決書查詢確認判決存在
Step 3: 產出結構化驗證報告
```

## 驗證報告格式

```
📋 法條引用檢查
├── ✅ [法規]第X條 — 正確
│    └── 🔗 {URL}
├── ⚠️ 引用[法規]第Y條 — 本案不適用
│    └── 扣 2 分
└── ❌ 引用已廢止條文
     └── 扣 5 分
```
