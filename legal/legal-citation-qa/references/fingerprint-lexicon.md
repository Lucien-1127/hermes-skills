# 語境指紋詞庫 — Context Fingerprint Lexicon

## 用途

抓到條文後，萃取關鍵詞與預期法域的指紋比對。如果匹配度不足，代表條文與法規名稱對不起來（pcode 錯誤 / 抓錯條）。

## 各法域核心指紋

### 民法指紋

等級 A（高辨識度 — 命中 ≥ 1 即強烈提示）：
損害賠償、債權人、債務人、繼承人、連帶責任、不當得利、侵權行為

等級 B（中等辨識度 — 需 ≥ 2 才確認）：
給付、契約、解除、終止、催告、時效、消滅

### 刑法指紋

等級 A：
有期徒刑、拘役、罰金、死刑、公訴、累犯、未遂犯、緩刑

等級 B：
犯罪、故意、過失、行為、結果、危險、不能、中止

### 公司法指紋

等級 A：
董事、監察人、股東會、董事會、股份、公司負責人、增資、減資

等級 B：
章程、決議、合併、分割、清算、重整、解散

### 憲法指紋

等級 A：
人民、權利、自由、選舉、基本國策

等級 B：
總統、行政院、立法院、司法院、考試院、監察院

### 實作比對邏輯

```python
FINGERPRINTS = {
    "民法": {
        "A": {"損害賠償", "債權人", "債務人", "繼承人", "連帶責任", "不當得利", "侵權行為"},
        "B": {"給付", "契約", "解除", "終止", "催告", "時效", "消滅"},
    },
    "刑法": {
        "A": {"有期徒刑", "拘役", "罰金", "死刑", "公訴", "累犯", "未遂犯"},
        "B": {"犯罪", "故意", "過失", "行為", "結果", "危險"},
    },
    "公司法": {
        "A": {"董事", "監察人", "股東會", "董事會", "股份", "公司負責人"},
        "B": {"章程", "決議", "合併", "分割", "清算", "解散"},
    },
}

def context_fingerprint(text: str, law_name: str) -> dict:
    fp = FINGERPRINTS.get(law_name)
    if not fp:
        return {"match": True, "score": 0.5, "matched_terms": []}
    matched_a = [kw for kw in fp["A"] if kw in text]
    matched_b = [kw for kw in fp["B"] if kw in text]
    score = (len(matched_a) * 2 + len(matched_b)) / (len(fp["A"]) * 2 + len(fp["B"]))
    match = len(matched_a) >= 1 or len(matched_b) >= 2
    return {"match": match, "score": round(score, 2), "matched_terms": matched_a + matched_b}
```

## 各法域禁忌詞

| 法域 | 禁忌詞 | 若出現代表 |
|------|--------|-----------|
| 民法 | 有期徒刑、拘役、罰金、死刑 | 很可能拿到刑法條文 |
| 刑法 | 損害賠償、不當得利、物權、繼承 | 很可能拿到民法條文 |
| 公司法 | 有期徒刑、拘役 | 拿到刑事責任條文（非公司法本文） |

## 擴充原則

- 新增法域時至少定義 A 級 5 個、B 級 5 個關鍵詞
- A 級關鍵詞若出現在錯誤法域的分析中，務必觸發警告
