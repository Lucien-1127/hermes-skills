# 自動攔截規則 — Interdict Rules

## 已知矛盾的條號-法規組合

### 絕對不存在（致命錯誤）

| 組合 | 建議修正 |
|------|---------|
| 民法第23條 | 可能混淆刑法第23條（正當防衛） |
| 民法第339條 | 可能混淆刑法第339條（詐欺罪） |
| 公司法第184條 | 可能混淆民法第184條（侵權行為） |
| 民法第271條 | 可能混淆刑法第271條（殺人罪） |
| 民法第320條 | 可能混淆刑法第320條（竊盜罪） |

### 存在但常誤用（需確認語境）

| 組合 | 注意 |
|------|------|
| 刑法第184條 | 不是侵權行為，是「損壞軌道罪」（公共危險罪章） |
| 刑法第339條 | 不是民法侵權，是「詐欺罪」 |
| 公司法第23條 | 包含第1項（對內）與第2項（對外連帶），引用須區分項次 |

### 跨法規交叉引用檢測

```
IF 產出同時包含「民法」且「第339條」或「第339-4條」:
  → 民法無詐欺罪章，339條系列均為刑法條文

IF 產出同時包含「公司法」且「第184條」:
  → 公司法無侵權行為條文，184條為民法條文
```

### 實作邏輯

```python
INTERDICT_RULES = {
    "hard_errors": {
        ("民法", 23): "民法無第23條，可能與刑法第23條混淆",
        ("民法", 339): "民法無第339條，可能與刑法第339條混淆",
        ("民法", 340): "民法無第340條",
        ("民法", 271): "民法無第271條，可能與刑法第271條混淆",
        ("民法", 320): "民法無第320條，可能與刑法第320條混淆",
        ("公司法", 184): "公司法無第184條，可能與民法第184條混淆",
    },
    "soft_warnings": {
        ("刑法", 184): "刑法第184條是損壞軌道罪（公共危險罪章），非侵權行為",
        ("刑法", 339): "刑法第339條是詐欺罪，注意與民法侵權行為區分",
    },
}
```
