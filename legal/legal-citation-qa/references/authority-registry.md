## 常混淆的組合（歷史錯誤記錄）

| 錯誤組合 | 正確對應 | 發生原因 |
|---------|---------|---------|
| 民法 + C0000001 | 民法 = B0000001 | 誤用刑法 pcode |
| 公司法 + J0030000 | 公司法 = J0080001 | pcode 記憶錯誤 |
| 民法 §23 | 不存在（刑法 §23 = 正當防衛） | 條號跨法規汙染 |
| 公司法 §184 | 不存在（民法 §184 = 侵權行為） | 條號跨法規汙染 |
| 民法 §339 | 不存在（刑法 §339 = 詐欺罪） | 條號跨法規汙染 |

## 查詢範例

```python
# ✅ 正確
pcode = AUTHORITY_REGISTRY["民法"]["pcode"]  # → B0000001
url = f"LawSingle.aspx?pcode={pcode}&flno=184"

# ❌ 錯誤
pcode = "C0000001"  # 這是刑法！
url = f"LawSingle.aspx?pcode={pcode}&flno=184"
# → 會拿到刑法第184條（損壞軌道罪）
```

## 注意事項

- law.moj.gov.tw 的 URL 參數 `flno` 支援 `339-4` 格式（連字號）
- 部分舊條號（如 §340、§345）已刪除，查詢會回傳「系統訊息」
- 條號範圍為概估，實際以 law.moj.gov.tw 為準
- 新制定或修正的法律可能新增 pcode，需定期更新此表
