## 四、TYPE-S 送件前檢查

### 法條引用
- [ ] 刑訴法第457條第1項（檢察官指揮裁量權）— 主力依據
- [ ] 刑訴法第467條（法定停止事由）— 誠實標示不符，再轉裁量
- [ ] 憲法第23條（比例原則）— 補強論述
- [ ] 刑訴法第484條（備位救濟）— 聲明異議保留

### 格式
- [ ] 書狀第一行有正式標題
- [ ] 字階正確：18>16>15>14 pt
- [ ] 行高 25-26pt，邊界四面 2.5cm
- [ ] 目標 2 頁，第三頁無單獨日期
- [ ] 頁碼在頁尾

### 證據
- [ ] 診斷證明書 1 個月內 + 「需專人照顧」
- [ ] 戶籍謄本
- [ ] 安置機構有具體進度（書面報價／預約確認函）

## 五、文件產生流程

```
1. 依本規格撰寫 Python 腳本（python-docx）
2. python3 script.py → 產出 .docx
3. libreoffice --headless --convert-to pdf file.docx → 產出 .pdf
4. 驗證：頁數、A4尺寸、字型嵌入
5. smtplib 寄送（PDF + DOCX 雙附件）
```
