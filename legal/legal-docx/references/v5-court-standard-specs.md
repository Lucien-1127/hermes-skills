# v5 法院標準版 — 最終核准規格

來源：司法院民事訴訟書狀規則第3條實測通過版本。

## 字型

- **Linux 安裝**：`sudo apt-get install -y fonts-cns11643-kai`
- **字型名稱**：`TW-Kai`（全字庫正楷體）
- **Windows 替代**：DFKai-SB（標楷體），docx 中設 DFKai-SB 可互通

## 格式參數（司法院書狀規則第3條）

| 項目 | 值 | 法源 |
|------|:---:|:----:|
| 紙張 | A4 (21.0 x 29.7 cm) | 書狀規則第3條 |
| 邊界 | 上下左右皆 **2.5 cm** | 第3條第1項第1款 |
| 字型 | 14pt（全字庫正楷體 TW-Kai） | 同條（14號以上20號以下） |
| 行高 | 固定 **25 pt** | 同條（25點以上30點以下） |
| 頁碼 | 頁面底端置中 | 同條第1項第3款 |

## 字階層次（v5 實際採用）

| 層級 | 大小 | 用途 |
|:----:|:----:|:------|
| L1 | 18pt 粗體 | 標題（暫緩執行聲請書） |
| L2 | 16pt 粗體 | 段標題（一、二、三、四） |
| L3 | 15pt 粗體 | 主旨句（為聲請…事） |
| L4 | 14pt | 內文、案號、基本資料、附件、簽章 |
| 頁碼 | 10pt | 頁尾 |

## 頁數目標

2 頁以內。第三頁若只有簽名日期 → 縮減段落間距重新排版。

## 發送前檢查順序（重要！）

產出後不可直接寄出，必須依序檢查：

```python
# 1. 產出 docx
# 2. 轉 PDF
libreoffice --headless --convert-to pdf 書狀.docx

# 3. 檢查頁數（目標 ≤ 2 頁）
import re
with open('書狀.pdf', 'rb') as f:
    boxes = re.findall(rb'/MediaBox\s*\[([^\]]+)\]', f.read())
    print(f'頁數: {len(boxes)}')  # 若 3 頁且第三頁僅簽名日期 → 重排

# 4. 檢查 A4 尺寸
    w = (v[2]-v[0])*2.54/72; h = (v[3]-v[1])*2.54/72
    assert abs(w-21) < 0.5 and abs(h-29.7) < 0.5, 'NOT A4!'

# 5. 檢查字型嵌入
    for m in re.finditer(rb'/BaseFont\s*/([^/\]]+)', f.read()):
        if b'KAI' in m.group(1) or b'TW' in m.group(1): print('字型 OK')
```

## 如果頁數超過 2 頁

1. 先減段落間距（sa 值遞減：4→2→1→0）
2. 附件可降至 13pt
3. 刪第二頁說明頁（另附 email）
4. 內文精簡（合併短句、刪贅字）

- 司法院官方標準下，14pt + 25pt 行高可容納約 350 字/頁
- 本聲請書含附件清單約 650 字 → 2 頁合理
- 全字庫正楷體 TW-Kai 嵌入 PDF 成功，LibreOffice headless 轉換無誤
