## 🚀 Usage

These skills are designed for **Hermes Agent** (by Nous Research).
Each `.md` file is a complete system prompt + workflow that can be:

1. **Placed directly** in `~/.hermes/skills/<category>/<skill_name>/SKILL.md`
2. **Loaded on demand** via `skill_view(name='<skill_name>')` in Hermes
3. **Auto-routed** via keyword triggers when SOUL.md contains the routing table

### Quick Install

```bash
# Clone the repo
git clone https://github.com/Lucien-1127/hermes-skills.git ~/hermes-skills

# Copy a skill into Hermes
cp -r hermes-skills/content-processing/core-summarizer ~/.hermes/skills/content-processing/
```
