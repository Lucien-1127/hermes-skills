# 申論題測試模組參考（ESSAY_TEST v1.0.0）

GitHub: `docs/40_模組與人格層/44_模組_申論題測試_v1.0.0.md`

## 三區溫度設定

| 區塊 | 溫度 | 原因 |
|------|:----:|------|
| 📝 出題 | 0.8 | 自然引導，去AI味 |
| 📊 批改總評 | 0.7 | 真人助教語氣 |
| 📋 法條引用判分 | 0.1 | 精準嚴謹 |

## 狀態機

S0_IDLE → S1_ASKING(出題) → S2_ANSWERING(作答) → S3_GRADING(批改) → S4_DONE

## 題型選項（法律版）

| 題型 | 範例 |
|------|------|
| 法條適用題 | 給情境，問適用哪條法 |
| 判例分析題 | 給判決摘要，問見解 |
| 爭點歸納題 | 給複雜案情，問爭點 |
| 策略題 | 問訴訟或答辯方向 |

## 人性化出題（去AI味）

避免「請說明…構成要件」等模板，改用：
- 「你覺得這樣算不算…？」
- 「如果換成你，你會怎麼判斷？」
- 「還記得法條怎麼規定的嗎？」

## 批改結構

1. 總評（🔥0.7）— 優點→建議，自然語氣
2. 逐題批改（🔥0.1法條引用）— 附來源URL
3. 給分標準：法條正確40% / 爭點掌握30% / 論理邏輯20% / 表達完整性10%
4. 全場引用驗證總報告

## 免責要求
