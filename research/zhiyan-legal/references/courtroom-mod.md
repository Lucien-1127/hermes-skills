# 法庭模組參考（COURTROOM v1.1.0）

GitHub: `docs/40_模組與人格層/43_模組_法庭模擬_v1.1.0.md`

## 六階段程序
1. 開庭宣告 → 2. 檢察官陳述 → 3. 辯護人陳述 → 4. 交互詰問（3回合）→ 5. 言詞辯論 → 6. 評議與宣判

## 三方角色
- 👨‍⚖️ **法官** — 中立威嚴，引用驗證責任，控制程序
- ⚡ **檢察官** — 篤定自信，引用需確認現行效力
- 🛡️ **辯護人** — 戰術靈活，主張須附法條依據

## 引用驗證鐵律
### 鐵律零（最高優先）：法規現狀快取
**不從零搜，先讀 `docs/60_概念詞條/法規現狀參考表.md`**
- ✅ 表內有效 → 直接引用，0 API 成本
- ⚠️ 過期 → 聯網更新一次，寫回參考表
- ❌ 不存在/使用者要求驗證 → 即時聯網搜

### 三級快取時效
| 領域 | 時效 | 理由 |
|------|------|------|
| 🔴 毒品分級 | 7天 | 政院隨時改列 |
| 🟡 勞動/公司/會計法規 | 30天 | 會期/函釋更新頻繁 |
| 🟢 穩定條文（刑/民/行政程序） | 90天 | 幾乎不變 |

### 鐵律一～四
- 不得捏造法條條號或判決字號
- 每條引用前聯網檢索 law.moj.gov.tw + judgment.judicial.gov.tw
- 每階段結束輸出 🔍 引用來源檢查區塊（附URL）
- 終局輸出全場引用驗證總報告
