# 建構邊界案例與資料清理

## 各分頁欄位差異

| 分頁 | 特殊情況 | 處理方式 |
|------|---------|---------|
| 憲法 | 第一行 header 為「完成之後，從試算表的第一頁開始檢查。」非 Unique_ID | build_index.py 過濾第一行，用 category+line_no 兜底 ID |
| 民法 | 多一欄 Debug_Log（第10欄） | parse_tsv 截斷為 9 欄 |
| 術語庫 | 3 欄結構（領域, 原文關鍵字, 建議標準譯名），非 9 欄 | 獨立 parse_terminology_tsv() |
| 全部 | 穿插編輯註記行（以「完成之後」「有關於」「另外」「如果你覺得」「請幫我」開頭） | is_data_row() 用 SKIP_PATTERNS regex 過濾 |

## 白話摘要覆蓋率

白話摘要並非所有條文都有。覆蓋狀況：
- **憲法/民法/刑法**：幾乎全覆蓋
- **小法及其他**：約一半有摘要（41,163 條，有摘要的約 30-40%）
- 無摘要時查詢回退顯示原始條文內容

## 重建注意事項

```bash
# 若 sheets 結構變動（新增/刪除欄位），需修改 build_index.py 的欄位對應
# 若新增分頁，需更新 TAB_CATEGORY 字典和 GID 對照
```
